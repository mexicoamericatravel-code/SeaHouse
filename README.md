# Casa Brisa del Mar

Clean upgraded version for GitHub + Netlify.

## Includes
- cleaner section backgrounds
- hero image slider
- clickable logo to top
- footer back-to-top button
- booking date pickers with nights + estimated total
- cleaner presentation for showing property owner
- extra page cards / section links
- multi-language UI
- multi-currency display
- Netlify-ready form

## Run
npm install
npm run dev

## Build
npm run build

## Netlify
Build command: npm run build
Publish directory: dist
