# Casa Brisa del Mar

Starter repo for a direct-booking room rental site near Cancun.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Netlify

- Build command: `npm run build`
- Publish directory: `dist`

## Edit later

Open `src/App.jsx` and replace:
- room names
- prices
- photos
- WhatsApp number
- email
- testimonials
- attractions
- add-ons
